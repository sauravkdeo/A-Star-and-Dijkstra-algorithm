

# C++ Dijkstra's and A* algorithm : Project 2
---

## Overview

C++ project to explore node using Dijkstra's or A* algorithm 


## Header/Libraries files

<map>, <iostream>, <queue>, <vector>,  <algorithm>, <math.h>, <opencv2> header files are used in the c++ code

cmake required to compile the program.


## Standard install via command-line and program run
```
unzip the files to a location
cd <path to repository>
mkdir build
cd build
cmake ..
make
Run program: ./app/Project2
```
